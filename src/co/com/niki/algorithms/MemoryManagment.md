# Memory management
## <b>Stack memory</b>
* Store active functions and local functions
* Function calls stored in stack memory 
* Reference to objects point to heap memory

## Heap memory</b>
* Slow access
* Store objects
* Garbage collector automatically remove references created in stack memory
